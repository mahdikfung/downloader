## 📝 معرفی

**StellarOracle** ابزاری ساده برای دانلود خودکار فایل‌ها از هر URL و نگهداری آن‌ها در GitHub است.

---

## ⭐ ویژگی‌ها

- ⚡ دانلود سریع و موازی
- 📦 تقسیم خودکار فایل‌های بزرگ
- 🔐 رمزگذاری و امنیت
- 🎯 استخراج خودکار لینک‌ها

---

## 🚀 شروع سریع

### مرحله 1️⃣: Fork کردن پروژه

**اول باید پروژه را fork کنید!**

1. به [ریپوزیتوری اصلی](https://github.com) بروید
2. دکمه **Fork** در بالا سمت راست را کلیک کنید
3. پروژه روی حساب GitHub خود کپی می‌شود

### مرحله 2️⃣: راه‌اندازی Workflow

1. به مخزن fork شده خود بروید
2. پوشه `.github/workflows/` را ایجاد کنید
3. دو فایل را داخل آن قرار دهید:
   - `Downloder.yml` - برای دانلود
   - `cleaner.yml` - برای پاکسازی

### مرحله 3️⃣: استفاده کردن

1. روی تب **Actions** کلیک کنید
2. **"⬇️ 01- Download from url"** را انتخاب کنید
3. **"Run workflow"** را کلیک کنید

---

## 📖 آموزش سریع

### 🎓 دانلود ساده

برای فایل‌های کوچک (کمتر از 45MB):

```
URL: https://example.com/file.zip
Mode: normal
```

---

### 🎓 دانلود فایل بزرگ

فایل به‌طور خودکار به اجزاء تقسیم می‌شود:

```
URL: https://example.com/large-file.iso
Mode: zip
```

---

### 🎓 دانلود با رمز عبور

برای محافظت فایل‌ها:

```
URL: https://example.com/file.iso
Mode: zip
Password: MySecurePass123
```

---

### 🎓 دانلود چند فایل

```
URLs: https://file1.zip https://file2.iso https://file3.exe
Mode: normal
```

URL‌ها را با **فاصلة خالی** جدا کنید.

---

### 🗑️ پاک کردن فایل‌ها

1. روی **"🗑️ 04- Clean downloads folder"** کلیک کنید
2. **"Run workflow"** را کلیک کنید
3. تمام فایل‌ها حذف می‌شوند

---

## 💡 نکات مهم

✅ **بهترین روش‌ها:**
- URL‌های صحیح و فعال استفاده کنید
- رمز عبور قوی انتخاب کنید
- مخزن خود را تمیز نگه دارید

⚠️ **دقت کنید:**
- حجم مخزن محدود است
- فایل‌های خیلی بزرگ timeout شوند
- رمز خود را با کسی به اشتراک نگذارید

---

## 🔧 مشکل‌گشایی

**دانلود شروع نمی‌شود:**
- URL را بررسی کنید
- اتصال اینترنت را تست کنید

**Timeout شده:**
- فایل را کوچک‌تر کنید
- Mode را روی "zip" بگذارید

**فایل دانلود نشده:**
- ERROR فایل را بررسی کنید
- دوباره سعی کنید
